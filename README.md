# PrepPilot-AI-Interview

PrepPilot AI is a full-stack web application designed to help students and job seekers prepare for technical and HR interviews through AI-powered mock interviews. The platform generates personalized interview questions based on the selected role, experience level, interview type, and difficulty, providing users with a realistic interview experience.

The application features secure user authentication, interview creation, AI-generated questions, answer evaluation, and detailed feedback to help users identify their strengths and areas for improvement. It also includes text-to-speech functionality, allowing interview questions to be read aloud for a more interactive experience, with support for both voice and typed responses.

This project was built to apply full-stack development concepts while exploring the integration of AI into interview preparation. It demonstrates practical implementation of authentication, REST APIs, database management, AI service integration, and responsive frontend development.

Features
Secure user authentication using JWT and bcrypt
AI-generated technical and HR interview questions
Personalized interview creation based on role and experience
Voice-assisted interview experience using Text-to-Speech
Support for both voice and typed answers
AI-powered answer evaluation with feedback
Interview and answer management
Responsive and user-friendly interface
